HTML Structure

This template is based on Twitter Bootstrap v2.2.2. It is fully redponsive. You can find detailed documentation about layout here.
Basically, template has only one column, except hidden box, which has two. But you can adjust it to any number of columns you need, from 1 to 12.

This is a standart layout with 1 column:

<div class="container-narrow"> <!-- Main container -->
  <header>
  </header>

  <div class="row-fluid"> 
    <div class="span12"> <!-- full width column -->

    </div>
  </div>

  <footer>
  </footer>

</div> <!-- / Main container -->

If you want 2 equal columns, your layout shoud look like this:

<div class="container-narrow"> <!-- Main container -->
  <header>
  </header>

  <div class="row-fluid"> 
    <div class="span6"> <!-- 6 columns - half of the width -->

    </div>

    <div class="span6">

    </div>

  </div>

  <footer>
  </footer>

</div> <!-- / Main container -->

Main container is responsive, by default it has 940px width and 90% max-width. You can adjust this values to your needs

.container-narrow {
  margin: 0 auto; 
  width: 940px; 
  max-width: 90%;
}

Themes

Template has two main themes – dark & light. There are separate CSS file for each of them - style_light.css and style_dark.css. To change background you need to edit body in css file of the theme you’ve chosen. This template has different background options:

Only color:

   body {
    ...
    background: #fff;
   }

Color with pattern:

   body {
    ...
    background: #1D113E url(data:image/png;base64,..somecode..);
   }

Note: All patterns included in online preview of this template are described in ../documentation/patterns.css file. You can find them and few more here or create your own here.

Full width image:

    body {
      ...
      background: url(../img/bg1.png) no-repeat center center fixed;
      -webkit-background-size: cover;
      -moz-background-size: cover;
      -o-background-size: cover;
      background-size: cover;
    }

Note: You can use one of the images included with this template (../img/bg) or grab your own. Be sure to use use large image, above 1700 pixels in width to cover almost 95% of the display sizes available today.

JavaScript

This template contains following jQuery plugins

  1. Header word resize
  2. Hidden info box
  3. Countdown
  4. Subscribe form
  5. Twitter feed

All JS code could be found in the end of index.php file. Origianl js files are located in ../js folder.

Header word resize

Since the template is responsive, the header word is also responsive. It’s done using FitText jQuery plugin - fittext.js. This is a simple setup:

 <script src="js/jquery.fittext.js"></script>
  <script>
   jQuery("#responsive_headline").fitText(0.6);
  </script>

Text will fluidly resize, by default: Font-size = 1/10th of the element's width. In this particular template it's set to 0.6. You can play with this number to find what fits your content better. For more complex tweaking you should see the documentation of the plugin.

Hidden info box

It's a simple jquery code to slide out hidden box.

 <script type="text/javascript">
  $(document).ready(function () {
  $("#info").hide();

  $("#info-link").click(function () {
    $("#info").slideToggle("slow");
    $(this).toggleClass("active"); return false;
  });
  });
  </script>

As You can see, #info is the hidden container's ID. And #info-link is the id of the trigger link.

Countdown plugin

Countdown here is reality thanks to - jQuery Countdown plugin Adjusting is realy simple. All you need is to put your launch date, for exampele the following denotes 10 December, 2013:

  <script type="text/javascript" src="js/jquery.countdown.min.js"></script>
   <script type="text/javascript">
   $(function () {
   $('#countdown').countdown({ 
    until: new Date(2013, 12 - 1, 10)});
  });
  </script>

JavaScript Date constructor expects the year, month, and day as parameters. However, the month ranges from 0 to 11. Thus You must leave -1 on it's place whatever date you pick up.

Subscribe form

This doesn't requre any database, since it's made on MailChimp API. All you need is to register a free MailChimp account if you don't have one already. Then just grab your API code and list id from it. You need to put them to inc/store-address.php. Also you can edit notification messages in the same file. Make sure that you save this file in plain text, otherwise you'll face an error. For more information about mailchimp API see the official documentation.

Twitter feed plugin

Here I've used jQuery live Twitter feed plugin. Simply put your username instead of dribbble in index.php file. For more tweaks see the documentation of the plugin.

  <script type="text/javascript" src="js/jquery.relatedtweets-1.0.min.js"></script>
  <script type="text/javascript"> 
    $(document).ready(function(){
      $('#tw').relatedTweets({
        debug:1
        ,from_users:'dribbble' // put here your twitter username
        ,status:1
        ,realtime:0
        ,n:20
        ,show_avatar:0
        ,show_date:0
        ,show_author:0
      });
    });
  </script>

Social icons

I've included in this template resolution independent social media icons made by me - Simplico. They are in two versions - for light & dark themes. They have hower effect with original color of particular social media on desktop and laptops. But on mobile they appear already with that color for better touch experience. That made using bootstrap's classes visible-desktop, visible-tablet and visible-phone. See index.php for comments.
